# Nook Browser

## Overview
An open-source browser project focused on simplicity, privacy, and customization. Nook aims to provide a lightweight browsing experience without sacrificing functionality.

## Technologies Used
- JavaScript/TypeScript
- Electron
- HTML/CSS
- GitHub CI/CD

## Key Features
- Borderless settings feature for distraction-free browsing
- Privacy-first approach with no telemetry collection
- Customizable UI with theming support
- Lightweight footprint compared to mainstream browsers

## Development Process
- Maintain separate dev and main branches
- Weekly pushes to main branch
- Collaborative development via GitHub
- Open source contribution model
